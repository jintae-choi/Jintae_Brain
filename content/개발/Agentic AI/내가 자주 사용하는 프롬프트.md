Claude Code 프롬프트 로딩 구조
계층	파일	로딩 시점	계정 이동 시
~/.claude/CLAUDE.md	전역 설정	매번	❌ 사라짐
프로젝트 CLAUDE.md	프로젝트 설정	매번	✅ Git과 함께 이동
MEMORY.md + 메모리 파일	자동 메모리	매번	❌ 사라짐
docs/*.md	참조 문서	필요 시만	✅ Git과 함께 이동
결론: docs/에 넣고 CLAUDE.md에서 참조만 하는 게 가장 효율적이고 이식성도 좋습니다.

CLAUDE.md = 이 프로젝트의 "사전" (매번 읽힘, 최소화)
docs/prompts/ = "필요할 때 꺼내는 참고서" (Git에 포함, 계정 무관)
메모리 = "보너스" (있으면 좋지만 없어도 docs/prompts/가 백업)



CLAUDE.md          → 이 프로젝트에서 매번 필요한 것만 (현재 상태 유지)
메모리 시스템       → 개인 프로필, 작업 선호 (이미 저장됨 ✓)
별도 프롬프트 파일  → 새 프로젝트 시작할 때만 붙여넣기
