# 문서 제목

## 개요
> 이 문서가 다루는 내용 한 줄 요약.

---

## 1. 전체 개념 / 흐름

개념의 큰 그림을 먼저 잡는다. 정의, 목적, 전체 아키텍처를 다이어그램/표로 표현.

```
[A] ──► [B] ──► [C]
         │
         ▼
        [D]
```

| 구성요소 | 역할 |
|---|---|
| A | ... |
| B | ... |

---

## 2. 사전 지식 (선택)

코드·명령어 이해에 필요한 기반 문법, 개념, 환경 설명. 불필요하면 섹션 삭제.

```bash
# 변수, 조건문, 반복문 등 기반 문법 예시
```

---

## 3. 전체 예시 코드

완성된 코드/명령어 전체를 먼저 제시한다. 조각이 아니라 "돌아가는 완성품"이 중요.

```bash
#!/bin/bash
# 실제 사용하는 명령어 또는 코드
```

---

## 4. 라인별 / 포인트별 상세 해설

위 코드의 각 구간을 잘라 전문용어·옵션·플래그 단위로 해설.

### 4-1. 환경 변수 / 초기 설정

```bash
VAR="value"
```

| 항목 | 설명 |
|---|---|
| `VAR` | ... |

### 4-2. 핵심 로직

```bash
command --flag arg
```

- `command` — 설명
- `--flag` — 설명
- `arg` — 설명

> **용어박스: 핵심 용어**  
> 낯선 전문용어가 나오면 인용 블록으로 분리해 설명.

---

## 5. 경험과 교훈 / 트러블슈팅 (선택)

### ❌ 실패: 증상 요약

```bash

```

**원인**: ...

**해결**:

